## 🧩 About This Repository

This repository contains structured and detailed solutions to vulnerable machines and CTF challenges from platforms such as:

- [VulnHub](https://www.vulnhub.com/)
- [Hack The Box](https://www.hackthebox.com/)
- [TryHackMe](https://tryhackme.com/)
- Custom-built labs and simulated environments

Each folder includes:

- 📝 Step-by-step walkthroughs  
- 🖥️ Exploit scripts and commands  
- 🧠 Privilege escalation techniques  
- 📄 Flags, proof of concepts, and lessons learned
